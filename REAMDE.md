# Praca magisterska

## Zależnośni
- node.js (16.3)[https://nodejs.org/en/download/releases/]
- python (3.8)[https://www.python.org/downloads/release/python-380/]

## Uruchomienie aplikacji

```bash
# cd backend
$ yarn install
$ yarn api

# cd prediction
$ pipenv shell
$ pip install -r requirements.txt
$ python application.py

# cd frontend
$ yarn install
$ yarn dev
```
